# VEDA_chatroom
채팅방 프로그램


## 서버 실행파일
```
git clone https://github.com/sm136599/VEDA_chatroom.git
cd VEDA_chatroom/server
mkdir build
cd build
cmake ..
make
./server
```

## 클라이언트 실행파일
```
git clone https://github.com/sm136599/VEDA_chatroom.git
cd VEDA_chatroom/client
mkdir build
cd build
cmake ..
make
./client <IP_ADDRESS>
```
